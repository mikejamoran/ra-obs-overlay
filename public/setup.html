Connect with Twitch
First-time setup (one-off)
  1. Go to dev.twitch.tv/console/apps/create
  2. Name: anything (e.g. "Stream Overlay")  ·  Category: Chat Bot
  3. OAuth Redirect URL:
  4. Click Create, then Manage
  5. Copy Client ID and generate a New Secret
  6. SSH to the server and add to ~/apps/ra-obs-overlay/.env: 
    TWITCH_CLIENT_ID=paste_client_id_here
TWITCH_CLIENT_SECRET=paste_secret_here
  7. Run pm2 restart ra-obs-overlay, then click Connect with Twitch above
Chat commands: Each Death Counter widget has a configurable command (default !death).
!death++ / !death-- / !death reset — permission level set per widget.
